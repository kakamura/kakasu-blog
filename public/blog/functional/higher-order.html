일급함수

함수를 출력하거나 값으로 다룰 수 있다, 즉 함수를 다른 변수와 동일하게 취급하는 것을 의미한다. 자바스크립트에서는 모든 것을 객체로 취급하기 때문에 함수 역시 예외가 아니다.

고차함수

따라서, '일급함수 지닌 언어'에서는 아래와 같은 방식 역시 가능하다.

  • 함수를 매개변수로 받아 사용
    • const apply1 = f => f(1);
const add2 = a => a + 2;
console.log(apply1(add2));
    const time = (f,n) => {
    let i = -1;
    while(++i < n) f(i);
    times(console.log, 3);
}
  • 함수를 반환하는 함수
    • addMaker = a => b => a + b;
const add10 = addMaker(10);
console.log(add10(5))      // 15
console.log(add10(10))     // 20
